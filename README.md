# 牛客网 Markdown 提取器

一键提取牛客网（NowCoder）竞赛题目内容，转换为标准 Markdown 格式并复制到剪贴板。

## 功能

- 在牛客网题目页面左侧标题栏注入「提取为 Markdown」按钮
- 自动提取题目名称、时间/空间限制、题目描述、输入/输出格式、示例数据和备注
- 准确识别 KaTeX 数学公式，转换为 LaTeX 行内公式（`$...$`）
- 自动处理富文本格式（粗体、斜体、代码块、列表、引用、图片等）
- 提取完成后自动将 Markdown 文本复制到系统剪贴板
- 绿色 Toast 提示操作结果

## 快速开始

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 开启右上角「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本项目所在的文件夹
5. 打开任意牛客网题目页面，点击标题栏旁的「提取为 Markdown」按钮

## 适用的页面

| 页面类型 | URL 示例 |
|---|---|
| 单个题目页 | `https://ac.nowcoder.com/acm/problem/*` |
| 比赛题目页 | `https://ac.nowcoder.com/acm/contest/*` |
